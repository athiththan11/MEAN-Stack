The above table presents all the books stored in our presented mLab (MongoDb). The edit button appends an edit form with the table row to edit the details of the books. Also, you can find the edit button with navigation to another page in Author's section.

The delete button deletes a book permenantly from the database. The ObjectID of the book is used to query and delete from the database.
